

# Яндекс Практикум, проектная работа "Посмотри В Окно"

## Описание

Данная страница позволяет "выглянуть" из окон самых разных городов мира.

## Автор

- Github - [VladaCode]([https://github.com/MikeBeloborodov](https://github.com/VladaCode/posmotri_v_okno)

## Функциональность

- Чекбокс с возможностью выбирать время суток.
- Возможность ввести в форму название интересующего города.
- Скроллбар с вариантами городов.
- Кнопка "Посмотреть еще" для продления списка городов.

## Технологии

- HTML
- CSS
- JavaScript
- API
